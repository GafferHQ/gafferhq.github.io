Requirements

Gaffer has the following requirements:

Operating System

  • Linux (most tested on CentOS 7 and AlmaLinux 9)
    • GCC 11 builds require distributions with glibc 2.28 or higher, and are not compatible with CentOS 7
  • Windows 10/11 (64 bit)
    • Installing the Microsoft Visual C++ Redistributable is required on Windows

Hardware

  • Nvidia GPU Recommended

3rd-Party Tools

Gaffer is compatible with the following tools:

  • Arnold 7.2 & 7.3
    • A minimum of 7.3.1.0 is required when using Arnold 7.3
  • 3Delight 2
  • Tractor 2
  • Deadline support can be enabled by installing the GafferDeadline extension
