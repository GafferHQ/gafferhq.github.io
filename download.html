Requirements

Gaffer has the following requirements:

Operating System

  • Ubuntu or CentOS Linux
  • OS X 10.9 or higher

Hardware

  • Nvidia GPU

3rd-Party Tools

Gaffer is compatible with the following tools:

  • Arnold 5
  • 3Delight 13
  • Tractor 2

Tinker


A good first step would be working through the Getting Started Tutorial in the user documentation. It covers the basics of working with Gaffer to create and render a scene, taking you from launching the application through to viewing a complete image.
